# Yewo

Application mobile POS offline-first pour café, bar, restaurant et snack.

## Stack
- React Native
- Expo 51
- Expo Router
- TypeScript
- Zustand
- Expo SQLite

## Mode de lancement actuel

Le projet utilise **Expo Router** comme point d’entrée principal.

Le lancement recommandé est :

```bash
npm install
npm run android
npm run start
